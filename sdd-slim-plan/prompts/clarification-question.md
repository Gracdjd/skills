# Clarification Question Template

Use this when asking about a single `Q*`.

```text
question: |
  Q1: <title>

  当前理解：<current understanding>
  为什么需要澄清：<why this blocks planning>
  受影响范围：<P*/T*/acceptance criteria>

  请只确认或修正这一点。
```

Rules:

- 一次只问一个 `Q*`
- 不要把多个疑问塞进同一条 askquestion
- 用户答复后要立即回写 spec
